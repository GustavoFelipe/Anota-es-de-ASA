Anotações:
============
  *Negrito: Cerca-se o texto como dois asteristicos (**)
  
  *Italico: Cerca-se o texto como um asteristico (*)
  
  Fonte Monoespacada duas crases (' ')
  
  Lista:
  ======
  Para cria uma lista.
  
   * Deixe uma linha em branco
   
   * Para cada item da lista
   
    * Iniciar com o asteristico (*)
    * Inicio teste
